# jorgeguimaraes.android Política de Privacidade Jorge Guimarães valoriza a privacidade dos seus usuários e se compromete a proteger seus dados pessoais. Esta política de privacidade descreve como coletamos, utilizamos e protegemos as informações dos usuários de nossos aplicativos para dispositivos móveis, incluindo as aplicações "Jorge Guimarães", "Restaurante Anastácia", "Arquivo.pt" e "Centro Clínico Santos Ferreira". Coleta e Uso de Dados Nenhum dado pessoal é coletado: Nossas aplicações foram desenvolvidas para oferecer funcionalidades específicas sem a necessidade de coletar dados pessoais dos usuários, como nome, endereço de e-mail, número de telefone, localização ou qualquer outra informação que possa identificar individualmente um usuário. Dados Anônimos e Agregados: Para fins de análise interna e melhoria contínua de nossos aplicativos, podemos coletar dados anônimos e agregados, como estatísticas sobre o uso da aplicação, mas esses dados não são vinculados a nenhum usuário específico. Segurança Embora não coletemos dados pessoais, adotamos medidas de segurança para proteger os dados que eventualmente possam ser armazenados localmente nos dispositivos dos usuários, como configurações de preferências ou dados de jogo. Transparência e Controle Acreditamos na transparência e no direito dos usuários de controlarem suas informações. Por isso, esta política de privacidade está disponível para consulta a qualquer momento.
